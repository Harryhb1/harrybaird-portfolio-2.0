GMTK GameJam 2020

Out of Space is a game I developed within 48hrs for the GMTK GameJam 2020 with the theme 'Out of Control'. The game consists of a user controlled space ship with various types of weaponry to fight off alien ships, the catch is you can't slow down! The result is a fast-paced intense video game which is fun to play. I developed the game using the Godot game engine and created all artwork completely independently.
